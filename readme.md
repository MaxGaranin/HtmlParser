# HtmlParser
Утилита для подсчета уникальных слов на заданной веб-странице.  
[Ссылка на репозиторий в GitHub](https://github.com/MaxGaranin/HtmlParser)

Принимает на вход один аргумент - адрес веб-страницы в полном формате, с http(s).  
Пример вызова: HtmlParser.ConsoleApp https://www.simbirsoft.com/

После запуска программа анализирует веб-страницу и выводит список уникальных слов в видимом содержимом на странице.

### Краткое описание алгоритма работы
Утилита скачивает веб-страницу, сохраняет ее на жесткий диск. Затем текстовый контент, который есть на ней (который видит пользователь на странице браузера), разбивается на слова и
подсчитывается количество нахождений на странице каждого уникального слова.
Разделителями слов считаются следующие символы: {' ', ',', '.', '!', '?','"', ';', ':', '[', ']', '(', ')', '\n',
'\r', '\t'}

В утилите учтено, что в случае если размер скаченной веб-страницы превышает
размер доступной памяти ОЗУ, то подсчет количества слов будет осуществляться корректно.
